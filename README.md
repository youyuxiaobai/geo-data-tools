# 地理数据爬虫工具集 v2.0

## 高德 POI 爬虫 (GaodePOISpider)
从高德地图 API 批量采集 POI（兴趣点）数据，支持按区域、类别等维度提取。

## OSM 数据下载器 (OSMDataDownloader) v2.0
从 OpenStreetMap 下载全球地理数据，导出为常见 GIS 格式。

### v2.0 新功能
- **国内/国外一键切换**：支持 35 个国家 + 158 个省/州/地区级联选择
- **启动动画**：粒子脉冲环炫彩开场
- 覆盖：亚/欧/北美/南美/大洋洲/非洲/中东主要国家

## 使用说明
- **高德 POI 爬虫工具**：见 `高德POI爬虫工具_项目介绍书.docx`
- **OSM 数据下载工具**：见 `OSM数据下载工具_项目介绍书.docx`
